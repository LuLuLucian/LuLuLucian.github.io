1.2 安装GitBook

1.2.1 安装NodeJs

由于存在兼容性问题，GitBook在高版本NodeJs上编译时会存在问题，所以需要安装NodeJs 12版本

nodejs v12下载

1.2.2 安装GitBook

打开命令行，windows是cmd或者PowerShell，执行以下命令安装GitBook：

npm install -g gitbook-cli

如果提示npm不是内部会外部命令，可以cd到node.js的安装目录在执行或将npm的目录添加到系统变量path中。
